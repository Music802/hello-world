#  NOTE 2:  #
#############
AVTech provides the below list of cgi-bin, its path structure should be:
	/cgi-bin/<user-class>/<command-cgi>
	where the <user-class> can be: nobody, guest, user, supervisor, power

GET  /cgi-bin/nobody/Captcha.cgi?action=get&image=login_bmp HTTP/1.1
